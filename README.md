# Bolão do Futebol — Apostador (versão simples, estável)
- Formulário do apostador (index.html) com 5 jogos e placares
- Geração de número de cartão **4 dígitos** com **contador local** (localStorage)
- Cartão pronto para print (card.html), com **logo à esquerda** e número **no canto superior direito**
- Sem banco de dados (ideal para primeira publicação na Vercel)

## Como usar
1. Adicione estes arquivos no repositório GitHub conectado à Vercel (raiz do repo).
2. A Vercel faz o deploy automaticamente. Abra `https://SEU-PROJETO.vercel.app`.
3. Preencha o formulário, clique **Gerar Cartão** — abre o cartão em nova aba.

> O contador local persiste por navegador/dispositivo. Para reiniciar, limpe o localStorage do site.
